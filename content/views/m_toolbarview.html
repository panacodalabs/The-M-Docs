M.ToolbarView

A view for showing a toolbar at the top or bottom of an application.
Basics · Screenshots · Properties · Methods · Example · CSS Styles

Basics

The toolbar view is a view to show a bar at the top or bottom of your application. In its basic variant, it only shows a centered label, which can be used to e.g. display the current page's name. But it also is possible to create more complex toolbars with buttons, button groups or images positioned at the left, center or right of the toolbar.

The toolbar view can contain up to three child views. Each child view must have a distinct, valid anchorLocation value. The valid anchorLocations for children of a ToolbarView are M.LEFT, M.CENTER and M.RIGHT.

Properties
Name Relevance Type Default value Description
anchorLocation optional String M.TOP Defines the position of the TabBar. Possible values are:
  • M.BOTTOM => is a footer bar
  • M.TOP => is a header bar
isFixed optional Boolean YES This property determines whether to fix the toolbar to the top / bottom of a page. By default this is set to YES.
showBackButton optional Boolean NO Determines whether to display an auto-generated back-button on the left side of the toolbar view or not.

Example

Setting up a default toolbar at the top of an application. 

M.ToolbarView.design({
    value: 'Toolbar'
})

Setting up a default toolbar at the bottom of an application. 

M.ToolbarView.design({
    value: 'Toolbar',
    anchorLocation: M.BOTTOM
})

Adding a button to the toolbar. 

M.ToolbarView.design({
    childViews: 'myButton label',
    myButton: M.ButtonView.design({
        anchorLocation: M.LEFT,
        value: 'Button'
    }),
    label: M.LabelView.design({
        anchorLocation: M.CENTER,
        value: 'Toolbar'
    })
})

Adding two buttons to the toolbar (left and right). 

M.ToolbarView.design({
    childViews: 'btnLeft label btnRight',
    btnLeft: M.ButtonView.design({
        anchorLocation: M.LEFT,
        value: 'left'
    }),
    label: M.LabelView.design({
        anchorLocation: M.CENTER,
        value: 'Toolbar'
    }),
    btnRight: M.ButtonView.design({
        anchorLocation: M.RIGHT,
        value: 'right'
    })
})

CSS Styles

The styling of the toolbar is done by jQuery Mobile. Depending on the anchorLocation property, some of the following css classes are responsible for the layout:

  • ui-header / ui-footer: This defines some basic layout and positioning setup for the toolbar.
  • ui-bar-a: This class defines the color and visual style of the toolbar.
  • ui-header-fixed / ui-footer-fixed: This indicates that the toolbar is fixed.
