M.MovableLabelView

A special LabelView that moves from left to right via css3 animation.
Basics · Screenshots · Properties · Methods · Events · Example · CSS Styles

Basics

The M.MovableLabelView inherits all methods and properties from the standard M.LabelView and extends it with special methods in order to make it movable via css3 animation. An internal calculation of the value/width checks if an animation of the label is necessary. If so an additional div container is created around this label and additional classes and styles are inserted into the DOM in order to make it move. For more information about the methods and properties of the standard M.LabelView please take a look at M.LabelView

Properties
Name Relevance Type Default value Description
movable optional Object NO This object must be passed to the label in order to make it movable. If not passed you have a standard M.LabelView. There are three properties available for movable:
  • time OR pxPerSec, one of these two must be set.
    time, for the label to move. The wider the label gets the faster it will move.
    pxPerSec, calculates the time for the animation by width/pxPerSec.
  • offset, (optional). If set it will add an offset to the left and right of the label div

Example

A MovableLabel with a "time: 10". By omitting offset it uses the standard value for offset. 

M.MovableLabelView.design({
    value: 'Make this slightly slightly slightly too big label move',
    movable: {
        time: 10
    }
});

A movable label with both pxPerSec and offset: 

M.MovableLabelView.design({
    value: 'Make this a movable label too',
    movable: {
        offset: 5,
        pxPerSec: 20
    }
});

CSS Styles

Based on the internal calculation additional styles are either removed or added. Extra styling is possible by passing classes to it via the cssClass property.
