Set it up

 You're just a few steps away from using Espresso on windows. We'll direct you to the pages where you find additional information instead of keeping redundant information here.
  1. Install Gow (GNU On Windows):
    https://github.com/bmatzelle/gow/downloads
  2. Install win-bash:
    http://sourceforge.net/projects/win-bash/files/latest/download?source=files
  3. Install node:
    http://nodejs.org/#download
  4. Download Espresso/The-M-Project SDK:
    http://the-m-project.org/
  5. Add Espresso DosKey (Unix alias equivalent): 
                    doskey espresso=node C:\Path\To\Espresso\bin\espresso.js $1 $2 $3 $4
    There's also a tutorial on how to persist the doskey so you don't have to enter it again and again on each start,here.
  6. Test the configuration:
    1. open command line (via shortcut if you followed the tutorial above for persisting the doskey)
    2. enter: 
      espresso version
    3. output should say: 
      Espresso 1.1.0
